===== VARIABLE SYSTEM =====

PMD uses mutable variables formatted as <lightness%>x<opacity%><+hue-offset>

Light Scheme OKLCH Mapping:
100x - Black/lowest priority (0% lightness) - oklch(0 0 0)
88x - Primary (28% lightness) - oklch(0.28 0.032 0)
88x+12 - PrimaryAux (complementary, +180° hue)
80x - Secondary (20% lightness) - oklch(0.2 0.032 0)
80x+12 - SecondaryAux (complementary)
64x -> 72x - Accent (56% in OKLCH) - oklch(0.56 0.094 0)
8x - Base (88% lightness) - oklch(0.88 0.056 0)
4x - Deep base (16% lightness) - oklch(0.16 0.022 0)
0x - White/highest priority (100% lightness) - oklch(1 0 0)

Dark Scheme OKLCH Mapping:
100x - White/highest priority (100% lightness) - oklch(1 0 0)
88x - Primary (88% lightness) - oklch(0.88 0.056 0)
88x+12 - PrimaryAux (complementary, +180° hue)
80x - Secondary (80% lightness) - oklch(0.8 0.1 0)
80x+12 - SecondaryAux (complementary)
64x -> 72x - Accent (72% in OKLCH) - oklch(0.72 0.122 0)
8x - Base (8% lightness) - oklch(0.2 0.032 0)
4x - Deep base (16% lightness) - oklch(0.16 0.022 0)
0x - Black/lowest priority (0% lightness) - oklch(0 0 0)

Note: OkLCH definitions use L (lightness) and C (chroma) values with hue = 0 for neutral tones.

DEPRECATED:
96x - Near-white (96% lightness) - oklch(0.96 0.016 0) [REDUNDANT - use 100x instead]

Each variable supports multiple opacity variants for different use cases.
Hue and chroma are globally mutable for theme customization.

Auxiliary hues use +180° offset by default: aux_hue = (primary_hue + 180) mod 360
