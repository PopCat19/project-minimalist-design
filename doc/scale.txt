===== SCALE SYSTEM =====

All measurements use pixel values at base 16px root font.
In CSS, implement with rem units (12px = 0.75rem, 16px = 1rem)
and scale root font-size for viewport size.

  Default (< 1200px): font-size: 16px
  Desktop  (≥ 1200px): font-size: 20px  (1.25× for FHD)

Borders:
  2px — Standard. Priority alerts, control clusters.

Outlines (inner, -offset):
  2px — Hover (100x). Highest priority, always distinguishable.
  4px — Focus-visible (80×48%). Medium weight for keyboard nav.
  Inner via outline-offset: -2px / -4px matching the width.

Gaps:
  4px  — Standard gap between sibling elements.
  8px  — Content group separation.
  10px — Dropdown chevron inner spacing.

Padding:
  4px block / 8px inline  — Buttons, bar widgets.
  8px all around          — Cards, notification panels.
  12px all around         — Flyout inner cards, player panels.

Negative Gaps:
  -8px  — Label stacked above dots in compact cards.
  -12px — Two-line drag handle indicator dots.
  For text-only vertical stacking. Never on interactive elements.

Icon Sizes:
  24px — Tab bar icons, media playback controls.
  20px — Dropdown chevrons, playlist controls.
  16px — Notification buttons, widget icons, action buttons.
  12px — Overlay icons on filled surfaces (drop one step).
