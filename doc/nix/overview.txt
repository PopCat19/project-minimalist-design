===== NIX INTEGRATION OVERVIEW =====

PMD provides Nix integration for NixOS through:
  1. Flake-based configuration
  2. Base16 export derivations
  3. Reproducible development shells

Core module: src/pmd-core.nix
  Exports dark/light variable sets, composite/stack/bake
  functions, mkBase16Slots, mkWallpaper, and mkScheme.

Build: nix build generates dark and light themes as base16
  YAML files and PNG wallpapers in share/pmd-base16/.

The Nix implementation mirrors the TypeScript code in
  src/ts/pmd/ — both use identical foundation slot values
  and compositing logic.
