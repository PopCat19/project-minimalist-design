===== NIX CONFIGURATION =====

Using PMD in a Nix flake:

  1. Add as a flake input:
       inputs.pmd.url = "github:PopCat19/project-minimalist-design";

  2. Access the scheme builder:
       pmd.lib.mkScheme { hue = 260; variant = "dark"; }

  3. Generate base16 YAML:
       nix build .#base16-export

  4. Access variables directly:
       pmd.lib.getVariables "dark"

The mkScheme function returns:
  { hue, variant, auxHue, variables, computed, base16 }

Where variables and computed contain the foundation slots
and derived surface/muted values respectively.
