===== PALETTE SYSTEM =====

PMD palettes are defined in YAML format with primary and auxiliary hues.

Example Structure:
scheme: "PMD [Color Name]"
author: "Project Minimalist Design"
hue: [0-360]
aux_hue: [0-360]  # Default: (hue + 180) mod 360

Auxiliary Hue Calculation:
Default: aux_hue = (primary_hue + 180) mod 360
Example: hue 345° → aux_hue = (345 + 180) mod 360 = 165°
Example: hue 260° → aux_hue = (260 + 180) mod 360 = 80°

Color Definitions:
base00-base07: Grayscale and neutral tones
base08-base0F: Accent and semantic colors

Each palette maps to PMD's lightness-based system:
100x, 88x, 88x+12, 80x, 64x, 8x, 0x

Multiple opacity variants are specified for each base color.
Auxiliary hues provide complementary contrast (+180° typically).
