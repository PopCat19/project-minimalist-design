===== PALETTE SYSTEM =====

Base16 YAML exports map PMD slots to the 16 base16 colour keys.

Mapping Table:
  Base16  PMD Stack     Description
  ──────  ────────────  ──────────────────────────
  base00  4x            True background (deepest)
  base01  8x            Base container
  base02  80×8%         Surface (composited)
  base03  80×48%        Muted / comments (baked)
  base04  72x           Subtext / secondary metadata
  base05  80x           Body text (main reading)
  base06  88x           Headers / strong text
  base07  100x          Max contrast text/symbol
  base08  88x           Alert / danger (root hue)
  base09  72x +290°     Constants (hue rotated)
  base0A  80x           Warning / classes (root hue)
  base0B  72x           Strings (root hue)
  base0C  80x +140°     Support (hue rotated)
  base0D  72x +30°      Functions (hue rotated)
  base0E  72x -30°      Keywords (hue rotated)
  base0F  80×48%        Deprecated / meta (baked)

  In light themes, accent slots (base09, base0B, base0D, base0E)
  use 72x luminance (0.32) with dark-mode chroma (0.122) to
  match the distinguishability of dark-mode accents. The PMD
  light 72x chroma (0.052) is intentional for subtext but too
  subtle for code accents. Gamut clamping handles edge hues.

Export Format:
  scheme: "PMD [Hue] [Variant]"  (e.g., "PMD 260 Dark")
  author: "Project Minimalist Design"
  base00–base0F: hex colour values (#RRGGBB, no prefix)
