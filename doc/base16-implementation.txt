===== BASE16 IMPLEMENTATION =====

The base16 palette is generated from PMD foundation slots via
the TypeScript module `src/ts/pmd/base16.ts` and the Nix module
`src/pmd-core.nix`. Both implement identical logic.

Slot Construction:
  - base00–base01: Direct foundation slot mapping
  - base02:        composited surface (80×8%)
  - base03:        baked muted (80×48%)
  - base04–base07: Direct foundation slot mapping
  - base08–base0E: Accent colours with hue rotation
  - base0F:        Same baked value as base03

Colour Pipeline:
  1. PMD OKLCH values → oklchToRgb → sRGB
  2. Gamut clamping via binary search (safeOklchToRgb)
  3. 8-bit hex encoding: #RRGGBB

Accent Lightness Override (Light Themes):
  Accent slots (base09, base0B, base0D, base0E) use PMD 72x
  luminance with bumped chroma (0.1) in light themes. The PMD
  light 72x chroma (0.052) is deliberately subtle for subtext
  but insufficient for distinguishable code accents. Dark mode
  uses the native PMD 72x chroma (0.122) without adjustment.
