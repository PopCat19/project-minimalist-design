# Context

- `ui-controls.ts` — Renders UI controls for color selection and theme presets
- `ui-docs.ts` — Manages documentation sidebar and doc loading in the UI
- `ui-export.ts` — Exports theme configurations to various formats
- `index.ts` — Barrel export module for all UI submodules
- `ui-render.ts` — Renders theme colors and UI components to the DOM
